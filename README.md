# NormalToolsProject
```
工具类项目
1、LoggerTools Log日志输出类
2、EmulatorUtils 模拟器检测类
3、PermissionUtils Android6.0（M）系统动态权限申请类
4、OnlySignUtils 移动设备唯一标识工具类
```
